💼 CareerMind-AI

An AI-powered Resume Analyzer that analyzes resumes, extracts candidate information, identifies technical skills and missing skills, provides personalized resume improvement suggestions, and generates AI-powered learning roadmaps to help candidates become job-ready.

🌐 Live Demo

🔗 https://careermind-ai-1uyb.onrender.com

✨ Features

* 📑 Upload resumes (PDF, DOCX, TXT)
* 🤖 AI-powered resume analysis with Google Gemini
* 🎯 Resume improvement suggestions
* 👤 Automatic extraction of candidate details
* 💼 Skills, Education, Experience & Projects extraction
* 📥 Download AI-generated analysis report
* 📱 Responsive and user-friendly interface

🛠️ Tech Stack

* Python
* Flask
* Google Gemini AI
* HTML
* CSS
* Bootstrap 5
* PyPDF2
* Python-docx
* FPDF
* Gunicorn
* Render
